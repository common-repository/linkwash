== Description ==

**Noch kein Linkwash Publisher? [Bitte registrieren Sie sich hier...](https://www.linkwash.de/register/)**

Linkwash wandelt die Links auf Ihrer Website automatisch und in Echtzeit in Affiliate-Links um. Mit nur einer Anmeldung können Sie so Tausende von Affiliate-Programmen bewerben und effizient Umsätze generieren.

* **Einfache Installation:** Aktivieren Sie das Linkwash Plugin, geben Sie Ihre Webseiten-ID ein und lassen Sie Linkwash fortan natürliche Links in Affiliate-Links umwandeln.

* **Mehr als 1.000 Partnerprogramme:** Mit Linkwash nehmen Sie automatisch an tausenden von Partnerprogrammen teil.

* **Mehr Zeit für Ihre Inhalte:** Dank Linkwash können Sie sich voll und ganz auf die Erstellung Ihrer Inhalte konzentrieren. Kein aufwendiges Anmelden für jedes Einzelne Partnerprogramm. Kein manuelles und fehleranfälliges Handling komplizierter Deeplinks.

* **Höhere Klickraten:** Die Umwandlung der Links in Affiliate-Links erfolgt „on-the-fly“ in dem Moment, in dem ein Nutzer auf einen Link klickt. Dadurch sehen die Links nicht nur natürlich aus, sie erzielen auch höhere Klickraten.

* **Transparente Statistiken:** Die umfassenden Auswertungen in Ihrem Linkwash Backend helfen Ihnen dabei, Ihre Inhalte gezielt weiter zu entwickeln.

Noch Fragen? In unseren [FAQ](http://www.linkwash.de/faq.html) finden Sie Antworten auf häufige Fragen zum Einsatz von Linkwash.

Überzeugt? Registrieren Sie sich [hier](https://www.linkwash.de/register/) als Linkwash Publisher und verdienen Sie Geld mit den Links auf Ihrer Website. Einfach, transparent und zuverlässig!

== Installation ==

1. Registrieren Sie sich als Linkwash Publisher. Hier können Sie sich anmelden: https://www.linkwash.de/register/

2. Laden Sie den Linkwash Ordner in das Verzeichnis /wp-content/plugins/ hoch.

3. Aktivieren Sie das Linkwash Plugin im „Plugins“-Menü in Wordpress.

4. Geben Sie Ihre Webseiten-ID ein. Diese finden Sie in Ihrem Linkwash Account unter „Websites“

Bitte beachten Sie: Sollten Sie den Linkwash-Code bereits manuell im Footer-Template hinterlegt haben, so entfernen Sie ihn bitte vor Aktivierung des Plugins.
